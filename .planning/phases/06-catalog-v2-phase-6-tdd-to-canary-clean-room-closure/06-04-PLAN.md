1. Before freeze: commit ALL of: any last pre-canary source fixes, R0–R3 receipt templates if tracked, 06-04-SUMMARY.md shell, 06-FINAL-REPORT.md shell, freeze verification script, and tracking docs.
2. Blocking human checkpoint confirms HEAD frozen and dirty tree excludes only intentional live evidence paths.

4. Live canary ledger/report fields are written to filesystem as uncommitted evidence. GSD phase completion uses committed shells + uncommitted live receipts; orchestrator must NOT auto-commit after canary.
5. If GSD executor would auto-commit: stop and hand off with classification recorded; do not violate freeze.
