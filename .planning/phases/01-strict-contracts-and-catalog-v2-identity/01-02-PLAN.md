## Interface contracts

- Signature: `validate_entity_graph_key(*, entity_type: str, graph_key: str, system_key: str) -> None` raises ValueError with stable code-ish message for grammar / system mismatch.
- Shell-level system_key only (no per-item override). Nested items receive system_key from parent model_validator context or parent field threaded via model_validator on the request that walks children.
- Discretion A1/A2: Oracle ident `[A-Z][A-Z0-9_$#]*`; overload `#` + non-empty `[A-Za-z0-9_$,#()]+`.
- edge_key remains bounded non-empty string; endpoint source/target graph keys use full grammar.
- Do not edit catalog_identity.py in this plan.
