## Interface contracts

```text
catalog_validation_error_to_structured(
    exc: ValidationError,
    *,
    correlation_id: str,
) -> dict[str, Any]
# keys: code, message, field_path, retryable, correlation_id
# code: CatalogErrorCode.validation_error or mapped unsupported_identity_schema / invalid_system_key when loc/type indicates
# message: bounded <=512, non-sensitive, no str(exc) full dump
# field_path: '.'.join(map(str, loc)) or None
# retryable: False for validation
```

Map when possible:
- identity_schema_version errors → unsupported_identity_schema
- system_key errors → invalid_system_key
- else validation_error

Do not require seven MCP wrappers if FastMCP already rejects before handler — unit tests call model_validate + converter + spy service methods directly. If graphiti_mcp_server has a natural single catch point without broad refactor, wire converter; else document unit-boundary sufficiency in SUMMARY.
